# 🍽️ Smart Canteen Ordering System

## Overview

The Smart Canteen Ordering System is a web-based application designed to simplify food ordering and canteen management. The system allows users to browse menu items, place orders, make payments, and track order status while enabling administrators to efficiently manage orders and menu items.

## Features

### User Module

* Browse available food items
* Add items to cart
* Place food orders
* Select payment method
* Track order status
* View order history
* Cancel orders before preparation begins

### Admin Module

* Manage menu items
* Add, edit, and remove food items
* Accept or reject orders
* Update order status
* View order history

### Cashier Module

* Confirm cash payments
* Manage payment records

## Project Workflow

1. User logs in using email or phone number.
2. User browses menu items.
3. User adds items to cart.
4. User places an order.
5. Payment is completed.
6. Admin receives the order.
7. Admin updates the order status.
8. User tracks the order until completion.
